# 🐾 Dolly's House — Tienda Virtual de Productos para Mascotas

> Proyecto académico desarrollado para el curso **Taller de Programación Web**  
> Universidad Tecnológica del Perú · Ciclo 05 · 2026

---

### ¿Qué es Dolly's House?

Dolly's House es una tienda virtual ficticia especializada en accesorios y alimentos para mascotas: perros, gatos, aves, hámsters y animales pequeños. El objetivo fue construir una presencia digital atractiva y funcional usando tecnologías web estándar.

---

### Páginas del sitio

- 🏠 **Inicio** — banner principal y productos destacados
- 🛒 **Productos** — catálogo por categorías con filtros visuales
- 🏢 **Sobre Nosotros** — misión, visión y datos de contacto
- 📬 **Contacto** — formulario de consultas y mapa integrado
- 📝 **Blog** — artículos sobre cuidado de mascotas
